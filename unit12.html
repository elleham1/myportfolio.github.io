UNIT 12 Software Development

In this unit, we were instructed to create a software program for jaguar land rover which will help thier sales team calculate currency exchange rate.

Unit 12 contain two assignments. Assignment one focuses on the purpose of two sample python program, and we were instructed to identify the tools and techniques used. Examples of the technique used were variables, functions, command word, subroutines, basic string handling, input prompts and data structures. Furthermore, we were assigned to review the qualities of at least one of the programming languages provided in the assignment brief. Reviewing their efficiency, usability, and performance and creating a flowchart to show how the program will run, whiles highlighting the strength and weakness. Open to view Assignment 1.

The second assignment covers the purposes and user requirement for the software program producing a design for a program highlighting the definition statement, solution alternative and a test data. Furthermore, we were assigned to fulfil the stated purpose developing a user interface constructs techniques and commentary throughout the code taking into the accountability of the code quality and user feedback..
